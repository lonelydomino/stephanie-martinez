# Stephanie Martinez — Spooky Creator

Premium personal brand site: Linktree replacement + full creator hub.

## Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide React + custom social icons
- Google Fonts: Cinzel + Poppins

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deploy to Vercel. Replace placeholder social/shop URLs and wire the newsletter form to Formspree or Resend. Add a real `public/media-kit.pdf` when ready.

## Customize

- Social & shop links: `src/components/Social.tsx`, `Shop.tsx`, `Navbar.tsx`, `Footer.tsx`
- Business email: `src/components/Collaborate.tsx`
- SEO: `src/app/layout.tsx`
